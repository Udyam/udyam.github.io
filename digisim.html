Introduction

Digisim basically stands for digital simulation. And digital simulation in layman terms can be expressed as the running of a digital systemâ€™s model. Digisim, is therefore a platform to explore and gain new insights into technology and to estimate the performance of systems without going into analytical solutions. Digisim is an event where participants have to simulate complex digital circuits. Participants will learn advanced circuit design and simulation.

Problem Statement:

Fascinated by contemporary processors like i7, i5 ,i3 etc. and want to explore their internal architecture ??
If yes , then Digisim is the event for you.
Participants will implement a subset of MIPS architecture (8 bit) with limited number of instructions in proteus.
MIPS(Microprocessor Without Interlocked Pipeline Stage) is a reduced instruction set computer architecture primarily used in embedded systems such as routers , video games like PSP, PS2 etc.

Instruction set is as follows

I Type instruction

Command Op-code Rs Ds Imm. Value
Beq r1, r3, immediate 0010 001 011 immediate
Addi r3, r4, immediate 0001 100 011 immediate
Bneq r1, r4, immediate 0011 001 100 immediate
Sw im(r2), r1, immediate 0100 010 001 immediate
lw r1, im(r2), immediate 0101 010 001 immediate

R Type instruction

Command Op-code Rs1 Rs2 Ds Func-code
Sub r1, r4, r5 0000 100 101 001 001
Add r1, r3, r6 0000 011 110 001 010
Or r1, r4, r5 0000 101 111 010 011
Slt r1, r4, r5 0000 010 011 001 101
Xor r1, r4, r5 0000 110 100 101 100

J Type instruction

OP CODE (1111) 4 BITS JUMP VALUE

Event Structure:

1. Each team consists of maximum of three participants.
2. Each team need to implement a base minimum number of instruction to qualify.
3. Teams will be judged on the basis of number instructions implemented by their architecture.
4. Points will also be awarded for innovation in design.
5. First year team will be judged separately for best first yearite.
